## 📦 Plugins Included

### 🗂️ DeluxeMenus
Custom GUI menus built for ZetrexMC server:

| Menu | Description |
|------|-------------|
| `profile_menu.yml` | Player profile menu with stats, balance, and rank |
| `kits_menu.yml` | Kits menu with Claim button and Cooldown display |
| `coin_shop.yml` | Coin shop (GoldShop style) with buy/sell system |

### 🛒 ShopGUIPlus
Full shop with 7 categories:

| Category | Items |
|----------|-------|
| ⚔️ Combat | Swords, Armor, Bow, Shield |
| 🪨 Blocks | Stone, Wood, Sand, Obsidian |
| 🍎 Food | Bread, Steak, Golden Apple |
| ⛏️ Tools | Pickaxes, Axes, Shovels |
| 💎 Ores | Diamond, Gold, Netherite |
| 🌿 Farming | Seeds, Wheat, Carrots |
| 🧪 Potions | Healing, Speed, Strength |

---

## ⚙️ Requirements

- DeluxeMenus
- ShopGUIPlus
- PlaceholderAPI
- Vault
- EssentialsX

---

## 📁 Installation

1. Copy `DeluxeMenus/` folder to `plugins/DeluxeMenus/menus/`
2. Copy `ShopGUIPlus/` folder to `plugins/ShopGUIPlus/shops/`
3. Run `/dm reload` and `/shopguiplus reload`
