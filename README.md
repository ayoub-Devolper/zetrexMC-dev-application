## 📦 Plugins Included

### 🗂️ DeluxeMenus
Custom GUI menus built for ZetrexMC server:

| Menu | Description |
|------|-------------|
| `profile_menu.yml` | Player profile menu with stats, balance, rank and navigation |
| `kits_menu.yml` | Kits menu with Claim button, Cooldown display and permission check |
| `coin_shop.yml` | Coin shop (GoldShop style) with buy system and rank purchasing via LuckPerms |

### 🛒 ShopGUIPlus
Full shop with **9 categories** and **170+ items**:

| Category | Items |
|----------|-------|
| 🛡️ Armor | Leather, Chainmail, Iron, Gold, Diamond, Netherite, Elytra |
| 🪨 Blocks | Stone, Wood (6 types), Sand, Concrete, Wool, Terracotta |
| 💎 Ores & Materials | Coal, Iron, Copper, Gold, Diamond, Emerald, Netherite |
| 🧪 Potions | Healing, Strength, Speed, Invisibility, Fire Resistance and more |
| 🏹 Combat & Weapons | Swords, Bows, Crossbow, Trident, Mace, TNT |
| 🌿 Farming & Nature | Seeds, Crops, Saplings, Bamboo, Kelp, Berries |
| 🎣 Misc & Special | Totem, Beacon, Shulker Box, Dragon Egg, Enchanted Books |
| 🍎 Food | Steak, Chicken, Golden Apple, Cake, Stews and more |
| ⛏️ Tools | Pickaxes, Axes, Shovels, Hoes (all tiers) |

### 👹 MythicMobs
Custom boss built for ZetrexMC server:

| Boss | Description |
|------|-------------|
| `HellKing.yml` | Hell King boss with 2000 HP, 5 abilities, rage mode at 30% HP and minion spawning |

**Abilities:**
- 🔥 Fire aura around the boss at all times
- ⚡ Lightning strikes every 5 seconds
- 💨 Launches nearby players into the air
- ☠️ Poison aura in 8 block radius
- 👹 Spawns 3 armed Hell Minions every 20 seconds
- 😡 Rage mode activated at 30% HP (Speed III + Strength II)

## 📁 Installation

### DeluxeMenus
```
Copy DeluxeMenus/ → plugins/DeluxeMenus/menus/
Run: /dm reload
Open: /profile | /kits | /coinshop
```

### ShopGUIPlus
```
Copy ShopGUIPlus/ → plugins/ShopGUIPlus/shops/
Run: /shopguiplus reload
Open: /shop
```

### MythicMobs
```
Copy MythicMobs/ → plugins/MythicMobs/mobs/
Run: /mm reload
Spawn: /mm mobs spawn HellKing
```
